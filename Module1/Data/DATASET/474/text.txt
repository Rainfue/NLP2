Цифровая трансформация российской экономики невозможна без увеличения числа IT-специалистов и повышения качества их обучения, которое должно максимально соответствовать
требованиям рынка труда. Одним из эффективных путей реализации практико-ориентированного
подхода к подготовке IT-бакалавров является разработка и использование в учебном процессе вуза
методов преподавания, которые в контексте смешанного обучения позволяют сочетать онлайн-курсы
с оффлайн-занятиями. Проведенный анализ условий эффективности использования онлайн-курсов
в высшем образовании позволяет построить оптимальную модель смешанного обучения в региональной университетской среде [3; 4]. Сравнение различных обучающих платформ выявило, что
интерактивные курсы HTML Academy являются одним из оптимальных вариантов для обучения
веб-разработке, поскольку они отвечают современным требованиям профессионального сообщества
к компетенциям студентов [1]. Разработанная методика обучения, основанная на использовании
интерактивных курсов HTML Academy, успешно применяется на факультете информационных
технологий и математики Нижневартовского государственного университета (НВГУ) и позволяет повысить качество профессиональных компетенций IT-бакалавров. Реализация смешанного
обучения, включающего онлайн-курсы, разработанные ведущими экспертами профессионального
сообщества, способствует преодолению разрыва между компетенциями выпускников IT-направлений и запросами работодателей, повышает качество образования в соответствии с требованиями
профессиональных стандартов.
Рассмотрим пример применения онлайн-курсов в НВГУ при обучении будущих IT-специалистов
веб-разработке. Формирование профессиональной компетенции студентов проводится в рамках расширенного подхода к обучению в области веб-разработки [5], основанному на широком привлечении работодателей и максимальном учете требований рынка труда на всех этапах проектирования и реализации
образовательной программы. Проектирование структуры и содержания профессиональной компетенции
осуществляется с использованием профессионального стандарта (ПС) «Разработчик Web и мультимедийных приложений» [2] и требований региональных работодателей к подготовке выпускников. В частности,
при проектировании профессиональной компетенции в области веб-разработки за основу были взяты
следующие трудовые функции ПС [2]: «Проверка и отладка программного кода», «Верстка страниц
информационных ресурсов (ИР)», «Кодирование на языках веб-программирования» и др.
Одним из недостатков подготовки выпускников вуза является неполное соответствие их компетенций реалиям современного уровня развития IT-технологий, а также недостаточный уровень
практических навыков. В силу объективных причин региональный вуз не обладает достаточными
собственными ресурсами для решения указанных проблем. Одним из инструментов, позволяющих повысить качество подготовки студентов в соответствии с современными требованиями рынка
труда, является использование онлайн-курсов, разработанных ведущими экспертами IT-отрасли.
Образовательная платформа HTML Academy представляет собой систему онлайн-курсов практической
подготовки обучающихся в области веб-технологий, разработанных ведущими представителями отрасли
в соответствии с современными IT-стандартами. В рамках программы академического сотрудничества
HTML Academy предоставляет учебному заведению доступ к личному кабинету преподавателя, который
может подключать к нему студентов и отслеживать прогресс прохождения ими онлайн-курсов (рис.).Курсы интегрированы в образовательный процесс в формате смешанного обучения. Структурно они включены в содержание лабораторных работ по дисциплине «Основы веб-разработки».
Перед выполнением онлайн-курсов студенты самостоятельно углубляют теоретические знания по
со­ответствующим темам дисциплины, полученные на обзорных лекциях.
Задания онлайн-курсов направлены на формирование у обучающихся практических навыков по
выполнению базовых типовых задач веб-разработчика. Каждое задание выполняется в интерактивной
среде, позволяющей видеть результаты выполнения кода в режиме онлайн. Это позволяет студентам
эффективно проводить анализ допущенных ошибок и оперативно исправлять их. Особую ценность для
формирования практических навыков веб-разработки представляют задания-испытания, в которых обучающимся предлагается выполнить типовые задачи по верстке веб-страниц и сравнить их с эталонными
решениями. На основе навыков, полученных при прохождении онлайн-курсов, студенты в рамках лабораторных работ выполняют проекты по созданию веб-страниц по заданному графическому макету.
Онлайн-курсы HTML Academy используются для формирования профессиональной компетенции веб-разработчика при освоении технологий HTML, CSS, JavaScript и PHP. В таблице приведено
соответствие между компонентами профессиональной компетенции в области веб-разработки,
осваиваемыми технологиями и применяемыми онлайн-курсами HTML Academy.
