Охарактеризован потенциал применения программной платформы Node.js для формирования корпуса текстов и дальнейшего статистического анализа на предмет частоты употребления определенной категории лексем. Предметом исследования стало сравнение процессов создания и работы программ, написанных на языках программирования JavaScript и Python , для сбора и статистического анализа текстовых данных. Рассмотрены преимущества платформы Node.js перед Python в аспектах парсинга JSON-данных и их дальнейшего анализа. В качестве примера применения JS-программ приведен анализ корпуса текстов сайта интернет-СМИ «Meduza» на предмет частоты употребления феминативов в последних десяти тысячах публикациях портала.
