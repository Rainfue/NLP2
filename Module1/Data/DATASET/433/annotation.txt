В работе рассматривается возможность представления гридсистемы в виде веб-сервиса с вычислительными узлами на основе веббраузеров, с применением технологии WebAssembly для повышения
производительности вычислений. Также приведен пример реализации
такой архитектуры с рассмотрением каждого компонента системы.
Описаны результаты тестирования реализаций вычислительных узлов
для программных средств: C# для десктоп приложения, JavaScript и
WebAssembly для веб-браузера.
