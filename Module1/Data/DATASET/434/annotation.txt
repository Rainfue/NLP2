В данной статье основное внимание уделяется изменениям в принципах реактивности между версиями фреймворка Vue.js. Определены критерии для сравнительного анализа реализации реактивности. В качестве предмета сравнения для каждой из исследуемых версий фреймворка разработано базовое приложение ToDo List, c помощью которого проводились измерения необходимые показатели и зафиксированы результаты.
