	
В статье обсуждаются результаты сравнительного анализа сетевых ресурсов для использования их в процессе обучения web-программированию учащихся средней общеобразовательной школы. Рассматриваются программные on-line средства разработки web-страниц (инструменты разработчика в браузерах, интегрированные среды разработки, фреймворки) в эргономико-технологическом, дидактическом и методическом аспектах. Кратко описываются основные характеристики сетевых сервисов и исследуются возможности их применения в учебном процессе для наглядной и доступной демонстрации учащимся процесса реализации основных web-технологий (HTML, CSS, Java Script). Приводятся рекомендации по отбору средств web-разработки для использования в процессе обучения школьников.
