Ресурсы сети Интернет давно и прочно заняли одну из
лидирующих позиций в качестве образовательного контента, дополнив, а
порой и заменив, учебники, учебные пособия и специализированные книги.
Образовательные ресурсы сети Интернет очень разнообразны. Помимо
электронных книг – это различные компьютерные обучающие программы,
автоматизированные учебные курсы, обучающие видеокурсы. Все более
прочные позиции в системе образования занимает такой вид как
дистанционное образование [1]. 
Целью статьи является теоретическое обоснование электронного
образовательного ресурса в виде обучающего сайта, посвященного
изучению одного из самых популярных языков программирования – Java. 
Изложение основного материала. Содержание сайта представлено
на русском языке. Навигационная панель (см. рис. 1) обеспечивает доступ к
следующим разделам сайта:
- «Главная страница» содержит краткую информацию о языке
программирования Java и актуальности его изучения для IT-специалиста;
- «Уроки» – это главный обучающий раздел сайта, открывающий
доступ к конкретным урокам, сгруппированным по темам. Раздел содержит
тесты для проверки и закрепления знаний по темам уроков;
- «Краткая история Java» знакомит пользователя с историей развития
языка программирования Java;
- «Интересные факты о Java»
Обучающий контент сайта сосредоточен в разделе «Уроки».
Материал сгруппирован по темам, большинство из которых заканчивается
предложением пройти тест. Страница с уроками показана на рис. 2. При
выборе конкретного урока происходит переход в соответствующий раздел
сайта. Материалы уроков взяты из [2].
Для прохождения теста необходимо перейти по гиперссылке. Тест
представляет из себя набор вопросов с альтернативным выбором. После
выполнения теста можно узнать результаты, перейдя по соответствующей
ссылке. На рис. 3 и 4 показан пример теста и результаты его выполнения.
При разработке обучающего сайта использовались различные
современные технологии веб программирования [3]. Разработаны элементы
сайта, отвечающие за навигацию, динамическое создание тестов, а также
проверка результатов тестов и отображение результатов на странице без
перезагрузки сессии. Макет сайта выполнен по технологии блочной верстки
стандарта html5. Разработана система каскадных таблиц стилей (CSS),
которая обеспечивает стилевое оформление сайта и интерактивность его
элементов.
На некоторых страницах с уроками имеются блоки текста, которые
посетитель сайта может скрывать или отображать по желанию. Чтобы
избежать ситуации, при которой на странице открыты одновременно
несколько блоков, был разработан скрипт на языке JavaScript с
использование одной из его библиотек (jQuery) [3]. Благодаря данному
скрипту на странице не может быть открыто сразу несколько блоков,
выделенных тегом details. 
 Таким образом, представленный в статье сайт может
служить в качестве вспомогательного обучающего ресурса по языку
программирования Java. Представленный на сайте материал позволяет не
только изучить язык, но и проверить уровень освоения материала с
помощью онлайн тестов.
