В статье рассмотрены преимущества и недостатки использования
популярного фреймворка Vue.js для создания пользовательских интерфейсов. Выполнен
сравнительный анализ с фреймворком React по популярности. Также представлен график
популярности по статистике запросов, которую предоставляет GoogleTrends. Дополнительно,
приведены активно развивающиеся библиотеки интерфейсов «BootstrapVue» и «Vuetify», их
популярность на основе запросов. Проанализирован способ создания приложений с помощью
различных библиотек
