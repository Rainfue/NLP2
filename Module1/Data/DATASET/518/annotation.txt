В ходе данной работы проведен эксперимент производительности приложений, написанных с помощью фреймворка Vue JS и библиотеки React JS (далее - фреймворки). Определены критерии эффективности рендеринга, на основании которых проведен сравнительный анализ. В качестве предмета сравнения для каждого из исследуемых фреймворков разработаны базовые приложения ToDo List.
