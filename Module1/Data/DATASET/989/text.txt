В наше время одним из популярным и полноценным языком программирования, 
который ориентирован на создание web – сайтов, считает PHP. Его распространённость 
обусловлена рядами преимуществ, который он даёт разработчику. Синтаксические 
особенности написания просты и понятны, благодаря схожести с языком 
программирование Си.
Возможности PHP очень большие. Область применения PHP сфокусирована на 
написание скриптов, работающей на стороне сервера. Например, обрабатывать данные 
формы, генерировать динамические страницы, работать с cookies и т.д. 
Главным фактором языка PHP является практичность. Данный язык 
программирования предоставляет программисту средства для быстрого и эффективного 
решения поставленных задач. Основные характеристики языка PHP.
1. Простота. Сценарий PHP может состоять из 1000 строк, а может из одной 
строки, всё зависит от заданной задачи. Не обязательно подгружать библиотеки, 
указывать специальные параметры компиляции или что-то в этом роде. Выполнение 
кода в PHP начинается с первой экранирующей последовательности (<?) и работает до 
того момента до того момента, пока не встретит парную экранирующую 
последовательность (?>). PHP – язык, который может быть встроен в HTML код страницы, 
который будет корректно обрабатывать PHP код.
2. Традиционность. Язык PHP будет знакомым тем программистам, 
работающие в разных областях. Многие конструкции схожи с языком Си или Pascal. Это 
заметно снижает начальные усилия для изучения.
3. Эффективность. Очень важное преимущество PHP заключается в том, что он 
не является ни компилятором, ни интерпретатором. Он является транслирующим 
интерпретатором. PHP позволяет обрабатывать сценарии с достаточно высокой 
скоростью. PHP вполне достаточна для создания вполне серьёзных web-приложений.
4. Безопасность. PHP предоставляет в распоряжение разработчиков и 
администраторов гибкие и эффективные средства безопасности, которые условно 
делятся на две категории: средства безопасности системного уровня и средства 
безопасности уровня приложения.
