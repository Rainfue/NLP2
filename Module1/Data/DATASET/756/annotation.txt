В работе описывается исследование возможности использования технологии webassembly совместно с react native для разработки кроссплатформенных мобильных приложений. Целью исследования является описание основных методов взаимодействия данных технологий. Основные задачи статьи заключаются в том, чтобы исследовать выбранные технологии, описать возможную архитектуру приложений, созданных при помощи данного стека и сделать вывод о том, можно ли использовать выбранные технологии для написания современных кроссплатформенных мобильных приложений. К сожалению, все эксперименты в статье не дали положительного результата, что привело к выводу, что на данный момент использование данных технологий затруднено. По результатам анализа предоставляется описание результатов экспериментов.
