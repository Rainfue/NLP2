Статья несет в себе информацию для PHP-разработчиков, целью которых является взаимодействие с API VK для загрузки медиа-контента на сервера VK (ВКонтакте - социальная сеть). В статье представлена последо- вательность реализации работающего кода, который можно будет использовать для добавления фотографий на стену, в альбомы или же в диалоги.
