Искусственный интеллект (далее по тексту ИИ) – это технология,
которая становится неотъемлемой частью жизни человека. ИИ можно
делегировать однотипные, трудоемкие и рутинные задачи. Более того, обучая
ИИ любому кейсу повышается качество работы, а количество ошибок сводится
к минимуму. Это происходит за счет исключения человеческого фактора, а
также из-за высокой вычислительной возможности современных систем.
Основой технологии ИИ считаются сверточные нейронные сети,
которые являются одними из наиболее используемых классов нейронных
сетей. Их можно задействовать в задачах исследования и анализа проблем в различных отраслях: промышленности, медицине, безопасности предприятий,
образовании, дорожном строительстве и других.
Нейронные сети разделяют на несколько базовых типов задач:
классификация, регрессия, прогнозирование, кластеризация, генерация. Задачи
классификации и кластеризации будут рассмотрены в предлагаемом
исследовании. Классификация используется для распознавания людей и
эмоций, геометрических тел, а также конкретных объектов из множества.
Кластеризация используется для анализа большого объема необработанных
данных для объединения данных по признакам.
При интенсивной эксплуатации зданий и сооружений с большой долей
вероятности возникают незначительные и существенные дефекты, которые
требуют экспертной оценки человека. В современном мире превалирует
высокоэтажная застройка, которую достаточно сложно оценить эксперту. Из-за
этого эксперт может пропустить важный дефект. При помощи ИИ работа
эксперта сможет быть эффективнее, так как ИИ будет выявлять дефекты на
зданиях, а человек оценивать его как существенный или малозначимый.
Для визуализации результатов анализа предметной области задачи
разработки технологии распознавания дефектов конструкций с
использованием искусственного интеллекта необходимо построить
функциональную модель процесса с декомпозицией на основные функции. Для
построения модели будем использовать методологию функционального
моделирования бизнес-процессов IDEF0. Основной целью является повышение
эффективности работы эксперта по оценке состояния фасадов на зданиях и
сооружениях. Процесс рассматривается с точки зрения разработчика.
Контекстная диаграмма функциональной модели процесса «Разработка
технологии по распознаванию дефектов конструкций на основе технологии
искусственного интеллекта» представлена на рисунке 1. 
Входными потоками является информация об объектах, их
характеристика и сформированный набор данных из изображений с
дефектами. Выходными потоками являются обученная нейронная сеть и
информация по перечню дефектов обследуемых объектов. В качестве
механизмов исполнения процесса предлагаются представители заказчика,
оборудование и эксперт в области технологии искусственного интеллекта при
решении задачи по разработке технологии распознавания дефектов
конструкций. Контролирующими и управляющими факторами являются
пожелания заказчика, требования авторского права и требования исполнителя.
Главная диаграмма функциональной модели процесса «Разработка

Первоочередной задачей исследования является обучение нейронной сети
распознаванию дефектов на зданиях и сооружениях.
Для обучения возьмем уже существующий набор данных,
распространенный в интернете (https://www.kaggle.com/datasets/arunrk7/surfacecrack-detection?resource=download) и добавим в него новые изображения.
Изображений должно быть достаточно много, чтобы изображений хватило на
все этапы задачи.
Все изображения следует разделить с наличием трещин и без наличия
трещин. Весь набор данных вмещает более 40 000 изображений размером 227
x 227 пикселей с каналами RGB. Для решения задачи будем использовать
готовые библиотеки на Pyton - Keras и TensorFlow. Все изображения из набора
данных будут разделены пропорционально для обучения, проверки и
тестирования. На рисунках 3 и 4 представлены изображения из набора данных.
При реализации задачи необходимо рассмотреть известные принципы
сверточных нейронных сетей:
• локальное восприятие;
• разделяемые веса;
• уменьшение размерности.
Относительно первой задачи можно сказать, что каждое изображение в
наборе данных будет разбиваться на области, которые составлены из пикселей.
У каждого пикселя есть интенсивность, каждое значение интенсивности будет
перемножаться с элементом ядра свертки. Нейронная сеть, исходя из предоставленного набора данных, самостоятельно определит, какие ядра
свертки нужны. Визуализация принципа локального восприятия представлена
на рисунке 5.
Второй принцип повествует о том, что ядро сверки для всех нейронов
одинаково. Таким образом, нейронная сеть ищет какие-то фрагменты в разных
участках изображений, которые нужны по задаче. За счет совместного
использования одних и тех же весов в ядрах свертки, существенно
уменьшается количество весов, которые нужно определить в процессе
обучения. Визуализация принципа разделения весов представлена на рисунке
6.
Для уменьшения размерности используются слои подвыборки:
усреднение, выбор максимального значения. Стоит отметить, что нам важен
больше факт наличия признака, чем знание места его точного положения на
изображении. Другими словами, важно само наличие дефекта на изображении,
чем его точное нахождение. Визуализация принципа уменьшения размерности
представлена на рисунке 7.
Нейронная сеть требует большой вычислительной мощности, однако это
нивелируется ее главным достоинством, а именно тем, что она автоматически
определяет важные признаки во время обучения из множества доступных
данных.
Таким образом, в результате полученного опыта использования
сверточных нейронных сетей при решении задачи выявления дефектов на
зданиях и сооружениях можно сделать вывод о возможности и необходимости
применения данной технологии искусственного интеллекта для дальнейшего
решения задач, связанных с обеспечением безопасности зданий и сооружений,
а также для поддержания благоприятного вида фасадов зданий.
