В статье рассмотрена архитектура системы
обработки больших данных, основанной на инструментах
Apache Hadoop, Apache Flume и Apache Spark.
Продемонстрировано применение разработанной системы
для хранения и анализа наборов данных, состоящих из
генерируемых событий в репозитории GitHub - крупнейшего
в мире веб-сервиса на базе системы контроля версий Git.
Выбраны метрики, с помощью которых проведена оценка
полученных результатов работы системы.
