До 50-60-х годов прошлого века компьютеры были слишком дорогими
машинами, предназначавшимися только для особых задач, и выполнявшими только одну задачу за раз. Языки программирования в это время были
разработаны в основном для научных вычислений. Однако, время шло и
благодаря техническому и научному прогрессу цены на компьютеры стали
падать, а их скорость всё больше увеличивалась. Примерно в это время
люди начали задумываться о создании языков программирования, отличавшихся от используемого до этого машинного языка. Такие языки стали
носить название высокоуровневые. В них мелкие операции, производимые
компьютером, были объединены в более крупные, высокоуровневые операции. Это было сделано для того, чтобы облегчить работу человеку, писавшему программу. Это дало начало развития программирования в целом.
На сегодняшний день существует более сотни языков программирования
высокого уровня различных областей применения.
PHP - это распространённый язык программирования общего назначения с открытым исходным кодом. Он разработан Расмусом Лердорфом и
выпущен в 1995 году. PHP специально сконструирован для веб-разработок
и его код может внедряться непосредственно в HTML (HTML - это код,
который используется для структурирования и отображения веб-страницы)
[1]. Другими словами, это серверный язык программирования.
Node.js - это JavaScript-окружение построенное на движке Chrome V8.
Он создан Райаном Далем и первоначально выпущен в 2009 году [3].
Node.js представляет собой платформу или инфраструктуру на основе
JavaScript для веб-приложений.
PHP применяется преимущественно для разработки веб-приложений, на
этом языке написаны популярные сайты, библиотеки для сайтов и фреймворки. Фреймворк - заготовка, готовая модель для быстрой разработки, на
основе которой можно дописать собственный код [2]. Node.js так же имеет множество библиотек и сайтов написанных на нём, в добавок к этому может использоваться для более широкой сферы задач, к примеру он может
быть использован при написании десктопных приложений или даже по
созданию кода для микроконтроллеров. Обе платформы имеют свои преимущества и недостатки, проанализировав которые, можно предположить
их дальнейшие пути развития.
Преимущества Node.js:
1. Использование пакетного менеджера NPM, который позволяет скачивать пакты библиотек намного быстрее, значительно ускоряя процесс
поиска нужной библиотеки;
2. Асинхронность работы, которая позволяет не блокировать основной
поток выполнения и выполнять действия, во время ожидания программы.
К примеру приложение может отправить запрос к какой-либо базе данных,
а во время ожидания ответа выполнить какие-либо другие действия.
Недостатки Node.js
1. По стандарту не использует все ядра системы;
2. Не предназначен для серверов, требующих высокой производительности, к примеру, для выполнения анализа большого количества данных;
3. Пакеты NPM могут быть небезопасными.
Преимущества PHP:
1. легок в освоении;
2. имеется огромное количество библиотек и фреймворков;
3. поддерживается большим количеством разработчиков;
4. может быть развёрнут почти на любом сервере, и имеет огромное
количество хостингов, использующих его.
Недостатки PHP:
1. Код PHP – смесь из двух языков самого PHP и HTML, которая усложняет нахождение искомой части кода. Данная проблема может проявить себя при разработке больших веб-сайтов;
2. Веб-сайты, написанные на PHP, могут иметь проблемы с безопасностью.
Обе платформы и по сей день активно развиваются, к примеру, появилось множество фреймворков для Node.js: Express, Angular, Meteor, в то же
время у PHP Laravel, Yii, CodeIgniter. PHP в будущем вероятно будет двигаться в сторону улучшения безопасности и увеличения строгости синтаксиса, для улучшения читаемости кода. Node.js вероятно будет развиваться
в сторону улучшения работы на многоядерных машинах и увеличения
производительности.
