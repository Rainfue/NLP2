В статье представлено описание интеграции базы данных MySQL с фреймворком Express.js. Express.js является одним из популярных фреймворков для разработки веб-приложений на Node.js, а MySQL - одна из популярных реляционных баз данных. Указывается, что платформа Node.js позволяет использовать базы данных обоих видов. В статье представлены как основные концепции работы с MySQL, так и более сложные аспекты, например, установка и настройка соединения, выполнение запросов на выборку и изменение данных. Показан пример простейшего приложения, созданного с помощью Ex-press js; приведены примеры запросов на обновление, удаление, добавление и выборку.
