Развитие информационных технологий оказывает влияние на все 
области жизни, в результате чего качественно меняются экономика, 
социальная сфера, структура общественных отношений [1, с. 21]. Информационные технологии в организациях присутствуют как необходимый атрибут технологии управления процессом производства товаров 
и услуг, экономического анализа и принятия управленческих решений 
[2, с. 17].
В данной статье рассмотрена автоматизация процесса отправления 
электронного сообщения на почту клиента при оформлении заказа на сайте и отправления сообщения менеджеру магазина с информацией о клиенте и о его заказе. Данная функция возможна благодаря использованию 
языка программирования PHP.
Язык программирования PHP может быть использован для написания скриптов, запускаемых на стороне сервера [3, с. 12]. Язык способен 
решать те же задачи, что и другие скрипты CGI, включая обработку данных формы HTML, динамическое создание страниц HTML. Другая область применения PHP – это создание сценариев, запускаемых из командной строки. Используя язык PHP, создают сценарии, которые выполняются независимо от веб-сервера и браузера на конкретном компьютере [4, с. 10]. Еще язык PHP может использоваться для создания клиентских приложений.
Рассмотрим автоматизацию процесса отправления электронного 
письма. Для начала на веб-сервере необходимо создать файлы HTML, 
JavaScript и добавить несколько блоков внутрь карт HTML и некоторые 
функции в карте JavaScript (рис. 1).
Этими блоками была создана форма, в которую клиент вносит информацию о себе: имя, электронную почту и номер телефона. Эта форма 
находится в корзине клиента вместе с его заказом (рис. 2).
Далее необходимо создать отдельный модуль PHP, так как это отправка письма и она доступна внешним пользователям, поэтому, в целях 
безопасности, следует отделить от основного кода. Данный файл получает код файла JavaScript Ajax запрос. Будут получены: имя пользователя, 
e-mail, номер телефона и корзина. В корзине находится ID товара и его 
количество. Необходимо открыть Json файл и взять его содержимое, 
потому что на сервер отправляется минимум необходимых данных 
(ID и количество).
После чего необходимо создать само сообщение, в котором есть основные данные клиента и сам заказ. Следует перевести код в удобочитаемый вариант (рис. 3).
Для отправки сообщения следует ввести несколько переменных: 
куда будет отправляться письмо, что будет отправляться и копия письма 
(рис. 4).
Теперь после оформления заказа письмо автоматически придет на 
почту менеджера, а копия письма придет на почту клиента (рис. 5).
Широкий спектр функций языка, его простой синтаксис и поддержка 
различных операционных систем и веб-серверов сделали язык PHP не 
только инструментом для быстрой разработки веб-сайтов, но и для методического построения сложных систем. Таким образом, была создана 
функция для автоматизации процесса оповещения, которая позволит ускорить и упростить работу.
